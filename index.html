Навигация и помощь

Вкладка ":Журнал оценок"

Введите необходимые данные для каждого столбца таблицы. Для добавления нажмите "Добавить". Для удаления - "Удалить". Чтобы сохранить таблицу в формате .csv, нажмите сохранить.

Вкладка "Табель успеваемости"

Чтобы посмотреть таблицу вам необходимо сначала загрузить файл с данными. После загрузки файла таблица автоматически загружается.

Вкладка "График успеваемости"

Чтобы построить график, необходимо убедиться что вы загрузили таблицу, после загрузки таблицы вы можете нажать кнопку "Построить график" после которого вам выведется график.

Вкладка "Помощь"

В данном разделе содержится полезная информация по навигации и использования программы для пользователя.

Вкладка "Загрузить файл"

Для загрузки файла нажмите на кнопку "Выберите файл". Откроется окно выбора файла, выбираете нужный файл в формате .csv, нажимаете "ОК", после которого начинается загрузка файла и загружается таблица.
